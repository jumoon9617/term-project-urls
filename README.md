# Welcome to Term Project!
↓Remix document
- 📖 [Remix docs](https://remix.run/docs)

## 開発環境

- Node v20.18.1
- Prisma v6.1.0
- Remix v2.15.2
- Postgresql(docker-composeに記載アリ)

## dev環境プレビュー

devサーバーの起動(port=3000)

```shellscript
npm run dev
```

## docker操作方法

コンテナ立ち上げ

```sh
docker-compose up -d
```

コンテナ停止

```sh
docker stop term-postgres
```

コンテナ再起動

```sh
docker start term-postgres
```
